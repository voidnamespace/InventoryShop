# InventoryShop API

API для управления товарами, заказами и пользователями.

## Структура проекта

- `src/` — основной проект API
- `tests/` — юнит-тесты
- `InventoryShop.sln` — решение

## Стек технологий

- .NET 9
- ASP.NET Core Web API
- Entity Framework Core
- JWT аутентификация

## Старт проекта

```bash
dotnet build
dotnet run --project src/InventoryShop.Api
